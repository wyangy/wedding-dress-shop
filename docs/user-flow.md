# User Flow

## Main Buyer Journey

1. The buyer lands on the homepage.
2. The buyer reads a short introduction about the bridal dress collection.
3. The buyer clicks Shop.
4. The buyer browses available dresses.
5. The buyer opens a product detail page.
6. The buyer checks photos, measurements, size, condition, and dress details.
7. The buyer clicks Book Consultation.
8. The buyer contacts the designer to ask questions, arrange a viewing, or discuss fitting and tailoring options.

## Key User Need

The buyer needs confidence before booking a consultation because wedding dresses require accurate fit, clear condition details, and trustworthy presentation.
