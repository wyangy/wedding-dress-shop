# User Flow

## Main Buyer Journey

1. The buyer lands on the homepage.
2. The buyer reads a short introduction about the bridal dress collection.
3. The buyer clicks Shop.
4. The buyer browses available dresses.
5. The buyer opens a product detail page.
6. The buyer checks photos, measurements, size, condition, and dress details.
7. The buyer clicks Request a Fitting.
8. The buyer goes to a fitting request page.
9. The page shows the selected dress image and key dress details.
10. The buyer enters their name, email, phone number, and message.
11. The buyer sends a fitting request.
12. The designer contacts the buyer later to confirm availability and next steps.

## Key User Need

The buyer needs confidence before requesting a fitting because wedding dresses require accurate fit, clear condition details, and trustworthy presentation.

## Important Note

Submitting the form does not confirm a fitting automatically. It only sends a request for the designer to review and respond to.
