# Wireframe Notes

The wireframe overview image is saved here:

## Planned Wireframes

* Homepage
* Shop / catalogue page
* Product detail page
* Request a Fitting page

## Homepage

The homepage should include:

* Header with logo and navigation
* Hero section with main dress image and call-to-action
* Featured dresses
* How It Works section
* Contact / enquiry section
* Footer

## Shop Page

The shop page should include:

* Page title
* Short introduction
* Dress grid
* Dress cards with image, name, size, and button to view details

## Product Detail Page

The product detail page should include:

* Large dress image
* Additional images
* Dress name
* Measurements
* Condition notes
* Description
* Request a Fitting button

## Request a Fitting Page

The fitting request page should include:

* Image of the selected dress
* Key details of the selected dress
* Short explanation that the fitting is not confirmed immediately
* Name field
* Email field
* Phone field
* Message field
* Send Fitting Request button
