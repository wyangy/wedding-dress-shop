# Wireframe Notes

## Planned Wireframes

- Homepage
- Shop / catalogue page
- Product detail page
- Contact / Book Consultation page

## Homepage

The homepage should include:

- Header with logo and navigation
- Hero section with main dress image and call-to-action
- Featured dresses
- How It Works section
- Contact section
- Footer

## Shop Page

The shop page should include:

- Page title
- Short introduction
- Dress grid
- Dress cards with image, name, size, and button to view details

## Product Detail Page

The product detail page should include:

- Large dress image
- Additional images
- Dress name
- Measurements
- Condition notes
- Description
- Book Consultation button

## Contact / Book Consultation Page

The contact page should include:

- Short consultation message
- Contact form or enquiry section
- Space for buyer questions
