# Visual Design Direction

The visual direction for this project is inspired by an elegant bridal archive / boutique catalogue style.

## Visual Reference

Reference image: docs/design-reference/bridal-archive-reference.png

## Overall Mood

The website should feel:

- Elegant
- Editorial
- Bridal
- Warm
- Soft
- Boutique-like
- Image-led
- Calm and luxurious
- Easy to use

The design should feel romantic, but not childish or overly decorative.

## Colour Direction

Use a warm neutral palette:

- Ivory / off-white background
- Soft stone borders
- Warm beige sections
- Rust / copper accent colour
- Black or near-black text for strong contrast

The accent colour should be used for important details, buttons, small labels, and links.

## Typography Direction

Use:

- Large serif-style headings for an editorial bridal feeling
- Clean sans-serif body text for readability
- Italic serif accents for emotional words if needed

The typography should feel more like a boutique magazine than a basic ecommerce site.

## Layout Direction

The layout should include:

- Large hero section
- Strong visual storytelling image
- Clear call-to-action buttons
- Featured dress cards
- Simple benefit / service icons
- Clear How It Works section
- Soft call-to-action section near the bottom

## Adaptation for This MVP

The reference design includes ecommerce elements, but this project is not an ecommerce website.

This project should not include:

- Cart
- Checkout
- Online payment
- Buy online wording

Instead, the main call-to-action should be:

- Shop Dresses
- Request a Fitting
- View Details

## User Flow

The main user flow remains:

Home → Shop → Product Detail → Request a Fitting

## Design Priority

The dress images should be the main focus.

The design should support the dresses and make them feel special, without distracting from the catalogue.
